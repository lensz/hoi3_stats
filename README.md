hoi3_stats
==========

Generates statistics to Hearts of Iron 3

##Preferences for standard resolution 1680x1050:
- HOI3 size: x=1666 y=922
- The overlay itself is at x=0 y=960 with a size of 1680px x 89px

##Dependencies:
- python 2.7
- tesseract-ocr
Python-libs
- pyHook (1.5.1)
- pyhk (included)
- scipy
- PIL
